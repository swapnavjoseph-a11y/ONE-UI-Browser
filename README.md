# One UI Browser 🚀

A sleek, premium, minimalist web browser inspired by Samsung's One UI design. Built completely in Python with PyQt6!

## ✨ Features
* **AMOLED Dark Theme:** Deep black backgrounds that are easy on your eyes.
* **Centered Design:** Pill-shaped search bar and quick-access widget cards floating perfectly in the middle of your screen.
* **Dashboard Widgets:** Single-click shortcut cards to open Google, YouTube, and GitHub instantly.
* **One UI Accents:** Beautiful rounded squircle corners and responsive blue hover effects.

## 🛠️ How to Run It on Linux
1. Open your terminal (`Ctrl + Alt + T`).
2. Make sure you have Python and PyQt6 installed by running:
   ```bash
   sudo apt update && sudo apt install python3-pip python3-pyqt6 python3-pyqt6.qtwebengine
   ```
3. Download the `browser.py` file from this repository.
4. Run the browser using this command:
   ```bash
   python3 browser.py
   ```
